# Vim

本项目是我的vim和neovim配置（vim仅支持部分功能）。

## 结构

本目录下有很多.vim文件，结构如下：

1. init.vim

    整个配置入口文件，引入了其他的.vim文件

2. plugins.vim

    插件配置文件，主要根据nvim和vim引入不同的插件安装文件：

    - plugins-common.vim

        vim和nvim都会生效的插件

    - plugins-vim.vim

        仅会在vim生效的插件。

    - plugins-nvim.vim

        仅会在nvim生效的插件

3. coc.vim

    coc的设置，引入了所有的coc目录下的.vim文件

4. coc-settings.json

    coc相关插件的配置

## 安装

在此项目的根目录下有脚本 `install.sh`，需要在目录中执行：

```bash
./install.sh vim
```

就会自动完成配置的安装。
